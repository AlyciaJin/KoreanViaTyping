How to Complete the Practice
  • Install a Korean keyboard on your mobile device or laptop.
  • Select the lesson you want to practice.
  • Each lesson has two phases:
    • Phase 1. See Korean and type Korean
    • Phase 2. See English and type Korean
  • Type the words and press 'Enter' to move to the next word.
  • Capture the last page that shows your completion with the code.
  • Note the time limits and focus on practicing efficiently.
